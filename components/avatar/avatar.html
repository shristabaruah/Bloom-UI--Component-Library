Avatar

Avatar is used to represent a user, and displays the profile picture, initials or fallback icon.

Profile Avatar

Profile Avatars can be created using the class 'avatar', inside the img tag.

Sizes Avatar

Size of the avatars can be changed by adding sm class for small size , md class for medium size , lg for large size class to the existing avatar class.
