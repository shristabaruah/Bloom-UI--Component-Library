Introduction to bloom UI

bloom UI is easy to use component library , which helps Developers to use best styles and components, by using pre-defined classnames. bloom UI helps you to quickly build your Project UI, by focusing much on Functionality, because we take care of your Designs.

Installation guide

In order to introduce bloom UI to your project, copy the link provided and paste it in the head tag of your HTML and access classes provided in the Docs on your elements.
